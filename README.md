# Lifeboard

Lifeboard is a personal life management windows playable app which tracks tasks, habits, budget, journal entries, meal plans and goals.

## Features
- Tasks
- Habits
- Journal
- Meals
- Goals
- Weekly Review

## How to use the app
1. Download Lifeboard.zip from releases
2. Extract the zip folder
3. Run the .exe (application) file

If Windows SmartScreen blocks it, click "More info" then "Run anyway".
